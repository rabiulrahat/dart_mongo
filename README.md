# How to run database?
 * **First Type**
 `>> systemctl enable mongod`
 * **2nd Type**
 `>>  mongosh`
 *
 $ >>  mongosh
 Then  you will get a url and prot number , use those in you code.

### Json file import command to mongodb
mongoimport --jsonArray -d <'databasename'> -c <'collection name'> --file <'./path'>
